# Portfolio Daniel Imanuel Manafe

Portofolio personal untuk menampilkan profil, pengalaman, proyek, dan prestasi sebagai mahasiswa Instrumentation & Control Engineering di Universitas Gadjah Mada.

## Fitur

- **Tampilan Modern** - Dark theme dengan efek glassmorphism
- **Animasi Interaktif** - Motion, hover effects, dan 3D elements
- **Timeline Otomatis** - My Timeline dengan auto-scrolling
- **Responsif** - Tampilan optimal di berbagai ukuran layar
- **Download CV** - Tombol download CV langsung dari halaman utama

## Teknologi

- React + Vite
- CSS3 (Custom properties, animations, 3D transforms)
- GitHub Pages (Deployment)

## Quick Start

```bash
# Install dependencies
npm install

# Development
npm run dev

# Build
npm run build
```

## Struktur Folder

```
Portofolio/
├── src/
│   ├── App.jsx      # Main component
│   └── index.css    # All styles
├── public/          # Static assets
├── early_setup/     # Documentation
└── dist/           # Production build
```
